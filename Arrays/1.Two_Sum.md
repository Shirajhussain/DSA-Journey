# 001. Two Sum

**Platform:** LeetCode
**Difficulty:** Easy
**Topic:** Arrays, Hash Map

---

## Problem Statement

Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to the target.

* Each input has exactly one solution.
* You may not use the same element twice.
* Return the answer in any order.

---

## Approach

Instead of checking every pair of elements (`O(n²)`), we use an **unordered_map** to store each number and its index.

For every element:

1. Calculate the required value:

   ```cpp
   diff = target - nums[i];
   ```
2. Check if `diff` already exists in the hash map.
3. If it exists, we have found the required pair.
4. Otherwise, store the current number and its index.

This allows us to solve the problem in a single traversal.

---

## Algorithm

1. Create an empty hash map.
2. Traverse the array.
3. Compute `diff = target - nums[i]`.
4. If `diff` exists in the hash map, return both indices.
5. Otherwise, insert the current element into the hash map.
6. Continue until the answer is found.

---

## Time Complexity

* **Time:** `O(n)`
* **Space:** `O(n)`

---

## Key Interview Points

* Uses **Hash Map** for constant-time lookup.
* Better than the brute-force `O(n²)` solution.
* Each element is visited only once.
* Works because the complement is searched before inserting the current element.
