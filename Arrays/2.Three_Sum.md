# 015. 3Sum

**Platform:** LeetCode
**Difficulty:** Medium
**Topic:** Arrays, Sorting, Two Pointers, Hashing

---

# Problem Statement

Given an integer array `nums`, return all the unique triplets `[nums[i], nums[j], nums[k]]` such that:

* `nums[i] + nums[j] + nums[k] == 0`
* `i`, `j`, and `k` are distinct.
* The solution set must not contain duplicate triplets.

---

# Approach 1: Brute Force

### Idea

Generate every possible triplet using three nested loops and check whether their sum is equal to `0`.

Since duplicate triplets may occur, sort the array first and store the answer in a `set`.

### Algorithm

1. Sort the array.
2. Generate every possible triplet.
3. If the sum is `0`, insert the triplet into a `set`.
4. Convert the `set` into a vector.

### Time Complexity

* **O(n³)**

### Space Complexity

* **O(k)** (`k` = number of unique triplets stored in the set)

---

# Approach 2: Hash Set

### Idea

Fix one element.

The remaining problem becomes a Two Sum problem.

Use a Hash Set to search for the required third element.

A `set<vector<int>>` is used to remove duplicate triplets.

### Time Complexity

* **O(n²)**

### Space Complexity

* **O(n)**

---

# Approach 3: Sorting + Two Pointers (Optimal)

### Idea

1. Sort the array.
2. Fix one element.
3. Use two pointers to find the remaining two elements.
4. Skip duplicate values to avoid duplicate triplets.

This is the expected interview solution.

### Time Complexity

* Sorting → **O(n log n)**
* Two Pointer Search → **O(n²)**

Overall:

**O(n²)**

### Space Complexity

* **O(log n)** (recursion stack used by `std::sort`)
* **O(1)** extra space if the sorting stack is ignored.

---

# Comparison

| Approach     | Time  | Space    |
| ------------ | ----- | -------- |
| Brute Force  | O(n³) | O(k)     |
| Hash Set     | O(n²) | O(n)     |
| Two Pointers | O(n²) | O(log n) |

---

# Key Interview Points

* Start with the brute-force approach.
* Improve it using the Hash Set technique.
* Finally, optimize using Sorting + Two Pointers.
* Always skip duplicate values in the optimal solution.
* The Two Pointer solution is the most preferred approach in coding interviews.
