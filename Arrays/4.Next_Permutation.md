# GFG - Next Permutation

**Platform:** GeeksforGeeks
**Difficulty:** Medium
**Topic:** Arrays, Two Pointers

---

# Problem Statement

Given an array representing a permutation, rearrange the numbers into the lexicographically next greater permutation.

If such a permutation does not exist, rearrange it into the smallest possible permutation (sorted in ascending order).

---

# Key Observation

A permutation can be increased only if there exists an element that is **smaller than an element to its right**.

The next permutation should be:

* Just larger than the current permutation.
* The smallest possible among all larger permutations.

---

# Approach

The algorithm consists of three steps.

### Step 1: Find the Pivot

Traverse from right to left and find the first index `i` such that:

```text
arr[i] < arr[i + 1]
```

This is called the **pivot**.

If no such index exists, the array is already the largest permutation.

---

### Step 2: Find the Next Greater Element

Again traverse from the right and find the first element greater than `arr[i]`.

Swap both elements.

---

### Step 3: Reverse the Suffix

The part after the pivot is always in decreasing order.

Reverse it to obtain the smallest possible arrangement.

---

# Example

### Input

```text
arr = [2,4,1,7,5,0]
```

### Step 1

Pivot

```text
2 4 1 7 5 0
      ↑
```

---

### Step 2

Next greater element

```text
2 4 1 7 5 0
        ↑
        5
```

Swap

```text
2 4 5 7 1 0
```

---

### Step 3

Reverse suffix

```text
7 1 0
```

becomes

```text
0 1 7
```

Final Answer

```text
[2,4,5,0,1,7]
```

---

# Algorithm

1. Find the pivot from the right.
2. If a pivot exists:

   * Find the first greater element from the right.
   * Swap them.
3. Reverse everything after the pivot.

---

# Time Complexity

* Finding pivot → **O(n)**
* Finding successor → **O(n)**
* Reversing suffix → **O(n)**

Overall:

**O(n)**

---

# Space Complexity

**O(1)**

---

# Key Interview Points

* The suffix after the pivot is always sorted in decreasing order.
* Reversing the suffix produces the smallest lexicographical arrangement.
* If no pivot exists, reverse the entire array.
* This is the optimal in-place solution with **O(n)** time and **O(1)** space.
