# Find Index of Closing Bracket

**Platform:** GeeksforGeeks
**Difficulty:** Medium
**Topic:** Stack, Strings

---

# Problem Statement

Given a string containing square brackets (`[` and `]`) and the index of an opening bracket `'['`, find the index of its corresponding closing bracket `']'`.

If no matching closing bracket exists, return `-1`.

---

# Approach

We use a **stack** to keep track of unmatched opening brackets.

1. Start traversing the string from the given opening bracket index.
2. Push every `'['` onto the stack.
3. Whenever a `']'` is encountered, pop one opening bracket from the stack.
4. When the stack becomes empty, the current index is the matching closing bracket.

This approach naturally handles nested brackets.

---

# Algorithm

1. Create an empty stack.
2. Traverse the string from `pos` to the end.
3. Push `'['` onto the stack.
4. Pop the stack whenever `']'` is encountered.
5. If the stack becomes empty, return the current index.
6. If the traversal finishes without emptying the stack, return `-1`.

---

# Time Complexity

* **O(n)**

---

# Space Complexity

* **O(n)**

---

# Key Interview Points

* Stack is used to match nested brackets.
* Supports multiple levels of nesting.
* Returns the index of the matching closing bracket.
* Returns `-1` if no matching bracket exists.
